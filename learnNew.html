What are the different ways to select an element in the DOM?

There are 5 waays to select an element in DOM. These are
1. getElementById()
2. querySelectorAll()
3. querySelector()
4. getElementsByClassName()
5. getElementsByTagName()

What is the difference between innerHTML, innerText, and textContent ?

1. innerHTML returns all the elements including spacing , tags and texts
2. innerText returns only text content excluding extra spacing from inside an html tag
3. textContent returns only text contents including spcing but without any tags from inside an HTML element
